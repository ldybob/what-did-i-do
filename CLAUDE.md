# what-did-i-do 설치 레포

이 레포는 설치용 템플릿입니다. 사용자가 대화를 시작하면 **먼저 설치 여부를 확인하세요**:
전역 설정(`~/.claude/CLAUDE.md`, 윈도우 `C:\Users\<사용자>\.claude\CLAUDE.md`)에
`# Weekly Log (업무 기록)` 섹션이 있는지 확인합니다.

## 미설치 상태라면

`SETUP.md`를 읽고 설치 인터뷰를 진행하세요.
설치가 끝나면 이 레포는 더 이상 필요 없다고 안내하세요
(시스템은 사용자의 전역 설정에 설치되며, 이 레포 폴더는 삭제해도 됩니다).

## 이미 설치돼 있다면

설치를 다시 제안하지 말고, 무엇을 도울지 물어보세요. 가능한 작업:
- **업데이트**: `git pull`로 템플릿 최신화 후, `SETUP.md`의 재설치 절차
  (기존 `# Weekly Log` 섹션을 사용자 확인 후 교체 — 기존 설정값은 유지)
- **제거**: `UNINSTALL.md`를 읽고 따르세요

## 주의

- **이 레포 안의 파일을 수정하지 마세요** (설치 상태를 레포에 기록하지 않습니다).
  사용자가 `git pull`로 업데이트를 받을 수 있어야 합니다.
